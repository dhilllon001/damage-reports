# Damage Reports

Apple-style fleet damage reporting and estimate workspace.

## Features

- Damage reports dashboard with stats, filters, responsive table + card layout
- Row-level **Estimate** action with line items, document upload, and USD conversion
- Report detail page (timeline, finance, photos, repairs)
- New damage report and repair action flows
- Pagination and mobile-friendly layout

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
